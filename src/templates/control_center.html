How to Run a ZfDash Agent

To manage a remote host, run a ZfDash agent on that machine:

If ZfDash is installed:
sudo zfdash --agent
Running from source:
sudo python src/main.py --agent
Full setup from source:
git clone https://github.com/ad4mts/zfdash.git
cd zfdash
uv sync
sudo .venv/bin/python src/main.py --agent
Note: The agent listens on port 5555 by default with TLS enabled. On first run, you'll be prompted to set an admin password.
